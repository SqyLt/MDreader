# MDreader

MDreader is a lightweight Markdown editor prototype inspired by Typora-style writing. It provides live preview, split editing, outline navigation, local file import/export, and draft persistence in the browser.

## Run

Open `index.html` in a browser.

## Features

- Live Markdown preview
- Edit, split, and preview views
- Heading outline navigation
- Formatting toolbar
- Open local `.md`, `.markdown`, and `.txt` files
- Save Markdown files
- Export rendered HTML
- Auto-save drafts in browser local storage
- Drag and drop a Markdown file into the window
- Interface language setting with English, Simplified Chinese, Traditional Chinese, and Japanese

## Files

- `index.html`: application shell
- `styles.css`: interface and rendered Markdown styling
- `app.js`: editor behavior and Markdown rendering
